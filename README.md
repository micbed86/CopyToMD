# 📝 Copy with Formatting (as Markdown)

> A sleek, powerful Chrome (and other Chromium-based browsers) extension that flawlessly converts selected web content into clean, well-formatted Markdown, straight to your clipboard.

Whether you're taking notes in Obsidian, writing documentation, or moving content to Notion, this extension ensures you capture exactly what you see, perfectly translating HTML and inline CSS into Markdown.

## ✨ Features

- 🚀 **High Fidelity Conversion:** Fully supports standard CommonMark and GitHub Flavored Markdown (GFM) — including tables, task lists, and strikethroughs.
  
- 🎨 **Smart CSS Recognition:** Intelligently parses inline CSS (like `font-weight` and `font-style`) to capture formatting even when sites (like Google Docs or Notion) don't use standard semantic HTML tags.
  
- 🔗 **Smart Link Stripping:** Need just the text? Easily configure the extension to copy formatting while ignoring URLs.
  
- ⌨️ **Dual Custom Shortcuts:** - **Primary Shortcut:** Copies everything natively.
  
  - **Secondary Shortcut:** Copies formatting but strips out all links (perfect for cleaner notes).
- ⚙️ **Modern & Elegant UI:** Features a beautifully designed, dark-themed settings modal with smooth animations and toggle switches.
  
- 🌍 **Internationalization:** Built-in English and Polish language support with auto-detection.
  

## 📦 Installation

Since this extension is in development (or not yet published to the Web Store), you can install it manually:

1. Download or clone this repository.
  
2. Open Google Chrome and go to `chrome://extensions/`.
  
3. Enable **"Developer mode"** (toggle in the top right corner).
  
4. Click **"Load unpacked"** and select the folder containing the extension files.
  
5. You're ready to go! 🎉
  

## 💡 Usage

1. Highlight any text or tables on a webpage.
  
2. Press your configured shortcut (Default: `Ctrl + Shift + C`).
  
3. A sleek notification will confirm the copy.
  
4. Paste your perfectly formatted Markdown anywhere!
  

### Settings

Click the extension icon in your Chrome toolbar to open the settings menu. Here you can:

- Enable/disable the extension.
  
- Change the UI language.
  
- Record custom shortcuts for both full-copy and link-stripped-copy.
  
- Customize the notification text and display duration.

# 📝 Skopiuj z formatowaniem (jako Markdown)

> Eleganckie rozszerzenie do przeglądarki Chrome (i innych przeglądarek opartych na Chromium), które bezbłędnie konwertuje zaznaczoną treść ze stron internetowych na czysty, świetnie sformatowany kod Markdown prosto do Twojego schowka.

Niezależnie od tego, czy robisz notatki w programie Obsidian, piszesz dokumentację, czy przenosisz treści do Notion, to rozszerzenie gwarantuje, że skopiujesz dokładnie to, co widzisz, perfekcyjnie tłumacząc kod HTML i wbudowane style CSS na język Markdown.

## ✨ Główne funkcje

- 🚀 **Wysoka wierność konwersji:** Pełne wsparcie dla standardu CommonMark oraz GitHub Flavored Markdown (GFM) — w tym tabel, list zadań (checkboxów) i przekreśleń.
  
- 🎨 **Inteligentne rozpoznawanie CSS:** Rozszerzenie potrafi odczytywać wbudowane style CSS (takie jak `font-weight` czy `font-style`), aby uchwycić formatowanie (np. pogrubienie i kursywę) nawet na stronach, które nie używają standardowych tagów HTML (jak Google Docs czy Notion).
  
- 🔗 **Pomijanie linków:** Potrzebujesz samego tekstu? Łatwo skonfiguruj rozszerzenie tak, aby kopiowało formatowanie, ale ignorowało odnośniki URL.
  
- ⌨️ **Dwa konfigurowalne skróty klawiszowe:** - **Główny skrót:** Kopiuje wszystko zgodnie z oryginałem.
  
  - **Dodatkowy skrót:** Kopiuje formatowanie, ale usuwa wszystkie linki (idealne do tworzenia czystych notatek).
- ⚙️ **Nowoczesny i elegancki interfejs:** Pięknie zaprojektowane, ciemne okno ustawień z płynnymi animacjami i nowoczesnymi przełącznikami.
  
- 🌍 **Wielojęzyczność:** Wbudowana obsługa języka polskiego i angielskiego z automatycznym wykrywaniem preferencji przeglądarki.
  

## 📦 Instalacja

Aby zainstalować rozszerzenie ręcznie (wersja deweloperska):

1. Pobierz lub sklonuj to repozytorium na swój dysk.
  
2. Otwórz przeglądarkę Chrome i wpisz w pasku adresu: `chrome://extensions/`.
  
3. Włącz **"Tryb dewelopera"** (przełącznik w prawym górnym rogu).
  
4. Kliknij **"Załaduj rozpakowane"** i wybierz folder zawierający pliki rozszerzenia.
  
5. Gotowe! 🎉
  

## 💡 Jak używać

1. Zaznacz dowolny tekst, kod lub tabelę na stronie internetowej.
  
2. Wciśnij swój skrót klawiszowy (Domyślnie: `Ctrl + Shift + C`).
  
3. Eleganckie powiadomienie potwierdzi udane kopiowanie.
  
4. Wklej swój idealnie sformatowany kod Markdown w dowolnym edytorze!
  

### Ustawienia

Kliknij ikonę rozszerzenia na pasku zadań Chrome, aby otworzyć menu ustawień. Możesz w nim:

- Włączyć/wyłączyć rozszerzenie.
  
- Zmienić język interfejsu.
  
- Nagrać własne skróty klawiszowe dla obu trybów kopiowania.
  
- Dostosować treść i czas wyświetlania powiadomienia o skopiowaniu.
